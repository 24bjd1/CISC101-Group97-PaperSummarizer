# CISC101-Group97-PaperSummarizer
LLM system prompt for a research paper summarizer application

System_Prompt.md
Include: Basic goal and guidelines for the system

Modules:
 01: Handles user inputs, transforms data into JSON format. Proceeds to identify possible edge cases.
 02: Creates loop to summarize research papers. Handles focus if key topics are present.
 03: Handles edge cases through conditional logic and control flow. 
 04: Generates output and organizes according to output rules.
 05: Ensures consistency and flow of output for usability in academic and research contexts.
 06: Extracts citations if requested by user. Important for academic settings.
